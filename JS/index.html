Hola Mundo! // En este proyecto se van a ir creando archivos con ejemplos de código para aprender JavaScript. Cada archivo se va a ir editando y actualizando con nuevos ejemplos y explicaciones. El objetivo es crear un recurso completo y actualizado para aprender JavaScript de manera fácil y accesible. // Asi se llama a un archivo de JavaScript desde el HTML. El atributo "src" se utiliza para especificar la ruta del archivo JavaScript que se desea incluir en el documento HTML. En este caso, se está incluyendo el archivo "03-constantes.js" que se encuentra en la carpeta "01-tipos". Al incluir este archivo, el código JavaScript contenido en él se ejecutará cuando se cargue la página HTML.
